## 📱 Available Applications

### **WeSplit: Check-Sharing App**
WeSplit is a simple yet practical check-sharing app built with SwiftUI. It helps users calculate how to split a check among multiple people and allows them to add a tip, making it a handy tool for everyday use.

### **GuessTheFlag: A Flag Guessing Game**
Guess the Flag is an engaging game that helps users learn about world flags. Players are challenged to identify which flag belongs to which country from three options, making it both fun and educational.

### **ViewsAndModifiers: Understanding Views and Modifiers in SwiftUI**
ViewsAndModifiers is a foundational project that delves into how views and modifiers work together in SwiftUI. It clarifies the mechanics of the framework, enabling developers to build custom view modifiers and containers while gaining insights into creating efficient and reusable components.

### **RockPaperScissors: Brain Training Game**
RockPaperScissors is a brain training game that challenges players to win or lose at rock, paper, scissors by selecting the correct move. Players score points for correct answers over 10 rounds, making it an engaging way to practice decision-making skills and reinforce SwiftUI concepts.

### **BetterRest: Sleep Prediction App**
BetterRest is a SwiftUI app that uses machine learning to help users determine the optimal bedtime based on their desired wake-up time and sleep duration. By integrating components like `DatePicker` and `Stepper`, the app provides personalized sleep recommendations, showcasing the effective combination of SwiftUI and Core ML.
